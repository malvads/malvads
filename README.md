I'm a software engineer. Started coding at 11 and got into game modding, reverse engineering, memory hacking, binary reversing, and network protocols. First person to bypass [Miniclip](https://miniclip.com/)'s Agar.io antibot system and the first to get bots working reliably on public servers after their patches. Somehow all this weird background turned into a career.

Along the way I reverse engineered stuff like [A3 ERP](https://www.wolterskluwer.com/es-es/solutions/a3erp) and [ICG Front-Hotel](https://www.icg.es/en/solutions/hotel/), got on the [Google Bug Hunters Hall of Fame](https://bughunters.google.com/), shipped some tools to [Kali Linux](https://www.kali.org/), contributed to [RPM](https://rpm.org/) and the [Apache Software Foundation](https://www.apache.org/), and worked on the [redborder](https://redborder.com/) Linux distribution for a while. Got a few Ruby gems out there too. Everything I build is open source. I just prefer knowing what's running on my machine. And yours.

These are just the ones that survived my terrible habit of not pushing to GitHub. RIP to the rest.
